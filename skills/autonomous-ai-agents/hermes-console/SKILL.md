# Hermes Console (Obsidian Plugin)

Run Hermes Agent in a tabbed terminal inside Obsidian. Bridges the current note context and cursor position to Hermes for context-aware assistance.

## Installation

1. In Obsidian: Settings → Community plugins → Browse → Search "Hermes Console"
2. Install and enable
3. Configure Hermes path in plugin settings

Or install manually from [GitHub](https://github.com/dannyshmueli/obsidian-hermes-console).

## Features

- Tabbed Hermes terminal inside Obsidian
- Selected note text auto-sent as context
- Cursor position awareness
- Background status alerts when agent runs finish
- Works alongside existing Obsidian skills

## Usage

Open via command palette: "Hermes Console: Toggle"
Or use the ribbon icon.
